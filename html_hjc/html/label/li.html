      HTML 与 XHTML 之间的差异

      在 HTML 4.01 中，li 元素的 "type" 和 "value" 属性是不被赞成使用的。 在 XHTML 1.0 Strict DTD 中，li 元素的 "type" 和 "value" 属性是不被支持的。
