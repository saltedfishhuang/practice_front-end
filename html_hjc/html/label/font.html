HTML<font>标签

定义和用法

<font>规定文本的字体、字体尺寸、字体颜色。

实例

规定文本的字体、大小和颜色

显示效果

HTML 与 XHTML 之间的差异

在 HTML 4.01 中，font 元素不被赞成使用。 在 XHTML 1.0 Strict DTD 中，font 元素不被支持。
