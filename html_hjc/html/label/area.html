定义和用法

<area>标签定义图像映射中的区域（注：图像映射指得是带有可点击区域的图像）。
area 元素总是嵌套在 标签中。
注释：<img>标签中的 usemap 属性与 map 元素 name 属性相关联，创建图像与映射之间的联系。

HTML与XHTML的差异

在HTML中≶area&tg;没有结束标签。 在 XHTML 中,≶area&tg;必须正确地关闭。

提示和注释

注释：≶img&tg;中的 usemap 属性可引用≶map&tg;中的 id 或 name 属性（由浏览器决定），所以我们需要同时向 ≶map&tg;添加 id 和 name 两个属性。
